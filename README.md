VECNA: AI-Driven Automation Orchestrator
O VECNA é a espinha dorsal do ecossistema de automação da Upra Ads. Ele atua como um orquestrador modular que conecta motores de extração de dados a agentes de decisão baseados em LLMs.

 Tecnologias Utilizadas
Linguagem: Python 3.x

Automação/Scraping: Playwright / Selenium

IA: OpenAI API / LangChain

Processamento de Dados: Pandas / Numpy

 Arquitetura do Sistema
Data Mining (EsAoN Engine): Módulo especializado em extração cirúrgica de leads, removendo ruídos e filtrando alvos de alta qualidade.

Contextualizador NLP: Processa os dados brutos e os transforma em "prompts" otimizados para a IA.

Sophia Agent: Interface de saída que realiza a abordagem personalizada com base no timing definido pelo orquestrador.

Nota: O código-fonte está passando por refatoração para implementação de novas camadas de segurança e escalabilidade.
